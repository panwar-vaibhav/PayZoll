# PayZoll - Next-Generation Crypto Payroll Platform

PayZoll is a Web3-powered payroll management platform that enables businesses to handle employee compensation seamlessly using cryptocurrency. Built with React, TailwindCSS, and blockchain technology.

## 🌟 Features

- **Dual Dashboard System**
  - Employer Dashboard for payroll management
  - Employee Dashboard for payment tracking and token management

- **Crypto Payroll Management**
  - Multi-currency support (ETH, USDT, USDC)
  - Automated payment scheduling
  - Real-time payment tracking

- **ESOP/RSU Management**
  - Token vesting schedules
  - Automated distribution
  - Real-time token value tracking

- **Analytics & Reporting**
  - Comprehensive payment analytics
  - Performance metrics
  - Token distribution insights

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/panwar-vaibhav/payzoll.git
   ```

2. Install dependencies:
   ```bash
   cd payzoll
   cd PayZoll\ Frontend/
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📁 Project Structure

```
src/
├── components/         # Reusable UI components
```
